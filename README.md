## Why SUPER Skills For AI Vibe Coding

Super Healthcare Wellness helps AI coding agents handle health and wellness work with clearer safety boundaries. Use it for workflow analysis, trend interpretation, wellness data patterns, care operations, and non-diagnostic guidance structures.

These skills are designed for **AI vibe coding**: fast, agent-assisted building where the AI needs strong domain context, practical workflows, guardrails, and implementation-ready outputs.

**SEO and discovery keywords:** healthcare workflows, wellness analysis, health data, health trends, care workflows, non-diagnostic guidance, wellbeing, AI vibe coding, agent skills.

## Install

Copy this folder into your agent's skills directory, then restart or reload the agent.

```bash
cp -R super-healthcare-wellness ~/.your-agent/skills/
```

Use it by name:

```text
Use $super-healthcare-wellness to help with this request.
```
